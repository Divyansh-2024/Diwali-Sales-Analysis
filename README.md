# Diwali Sales Analysis (EDA Project)

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a Diwali sales dataset to understand customer purchasing behavior and uncover meaningful business insights. The analysis helps identify key factors influencing sales during the festive season.

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

## Key Analysis Performed

* Data Cleaning and Preprocessing
* Handling Missing Values
* Exploratory Data Analysis (EDA)
* Gender-wise purchasing analysis
* Age group analysis
* State-wise sales performance
* Product category analysis

## Key Insights

* Married women in the age group 26–35 are the highest buyers
* Certain states contribute significantly to total sales
* Specific product categories have higher demand during Diwali
* Most customers belong to specific occupation groups

## Visualizations

The project includes multiple visualizations such as:

* Bar charts
* Count plots
* Heatmaps
* Distribution plots

(All visualizations are created using Matplotlib and Seaborn)

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone <your-repo-link>
   ```
2. Navigate to the project directory:

   ```
   cd Diwali-Sales-Analysis
   ```
3. Install required libraries:

   ```
   pip install -r requirements.txt
   ```
4. Run the Jupyter Notebook:

   ```
   jupyter notebook
   ```
